# Horoscope API

Horoscope is a simple tool for getting horoscope data. It returns the horoscope for the day.

[![npm version](https://img.shields.io/npm/v/@apiverve/horoscope.svg)](https://www.npmjs.com/package/@apiverve/horoscope)

This is a Javascript Wrapper for the [Horoscope API](https://horoscope.apiverve.com?utm_source=npm&utm_medium=readme)

---

## Installation

Using npm:
```shell
npm install @apiverve/horoscope
```

Using yarn:
```shell
yarn add @apiverve/horoscope
```

## Usage

---

### Perform Request

Using the API is simple. All you have to do is make a request. The API will return a response with the data you requested.

```javascript
var query = {
  sign: "aries"
};

api.execute(query, function (error, data) {
    if (error) {
        return console.error(error);
    } else {
        console.log(data);
    }
});
```

---

### Using Promises

You can also use promises to make requests. The API returns a promise that you can use to handle the response.

```javascript
var query = {
  sign: "aries"
};

api.execute(query)
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error(error);
    });
```

---

### Using Async/Await

You can also use async/await to make requests. The API returns a promise that you can use to handle the response.

```javascript
async function makeRequest() {
    var query = {
  sign: "aries"
};

    try {
        const data = await api.execute(query);
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}
```
